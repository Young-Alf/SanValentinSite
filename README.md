# San Valentín Site

Un sitio web romántico con login por contraseña, galería de fotos, reproductor de música, dedicatoria y un juego interactivo. Perfecto para celebrar el Día de San Valentín.

## Cómo usar:
1. Clona el repo: `git clone https://github.com/tuusuario/SanValentinSite.git`
2. Abre `index.html` en un navegador o sube a GitHub Pages.
3. Agrega tus fotos en `images/` y música en `audio/`.

¡Feliz San Valentín! ❤️
